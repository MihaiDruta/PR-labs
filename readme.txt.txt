Paaii efectuati pentru lucrarea de laborator nr.1

1.Crearea unui repozitoriu pe github.com
Repozitoriul are denumirea PR-labs 
Link-ul catre repozitoriu: https://github.com/MihaiDruta/PR-labs

2.Crearea unui directoriu in care initiez repozitoriul local
Comenzile utilizate
mkdir lab1pr - pentru crearea directoriului
cd lab1pr - pentru setarea directoriului curent

3.Initializarea repozitoriului local
Comenzile utilizate
git init - pentru initializarea repozitoriului git local

4.Crearea fisierului readme.txt in care se afla informatia despre pasii efectuati

5.Adaugarea si commit-ul file-ului readme
Comenzile utilizate
git add readme.txt
git commit -m "Readme commit"

6.Crearea legaturii intre repozitoriul local si cel distant pe github.com
git remote add origin https://github.com/MihaiDruta/PR-labs.git
git remote -v

7.Push-ul nemijlocit a fisierelor din repozitoriul local pe cel distant pe github.com
